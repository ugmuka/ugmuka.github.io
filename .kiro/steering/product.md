# Product Overview

Data Engineer 向井雄二の個人ポートフォリオサイト。GitHub Pages 上で動作し、経歴・スキル・資格・登壇履歴を公開する。

## Core Capabilities

- **プロフィール表示**: 名前・職種・SNSリンク（X / LinkedIn / Qiita / Zenn / SpeakerDeck）
- **職歴・プロジェクト一覧**: 2020年以降のデータエンジニアリング実績
- **スキルマトリクス**: 言語・フレームワーク・クラウド・データ基盤ツール・BIツール
- **資格**: AWS / GCP / Snowflake 等の認定資格
- **登壇履歴**: テーブル形式でイベント・タイトル・資料リンクを掲載

## Target Use Cases

- 採用担当者・クライアントがエンジニアのスキルセットと実績を確認する
- データエンジニアリングコミュニティへの情報発信

## Value Proposition

サーバーレス構成により維持コスト不要。GitHub Pages にデプロイするだけで動作する静的ポートフォリオ。
